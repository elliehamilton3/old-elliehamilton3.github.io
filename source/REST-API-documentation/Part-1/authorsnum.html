authors/<author-id>

URI Template

http://waisvm-rfp1.ecs.soton.ac.uk/authors/<author-id>

URI Parameters

author-id

required

The URI Parameter <author-id> takes a number which corresponds to the id assigned to the author. This is automatically assigned by the system to be the first available id in the system which is not already being used. This allows for id's which may have been freed up following the delete of an author.

Example value: 116

HTTP Methods

GET, PUT, PATCH, DELETE, HEAD, OPTIONS

Response Formats

Renders Formats XML, JSON, TEXT/CSV
Parses Formats XML, JSON, TEXT/CSV
Requires Authentication? Yes - for PUT, PATCH and DELETE requests

DELETE /authors/116

Deletes the author located at /authors/116 if the user meets the required permissions i.e. if they are the owner of the author/they created the author.

HTTP Status Code: 204 NO CONTENT or if the correct authorization token id not given then returns 403 FORBIDDEN

publishers/<publisher-id>

URI Template

http://waisvm-rfp1.ecs.soton.ac.uk/publishers/<publisher-id>

URI Parameters

publisher-id

required

The URI Parameter <publisher-id> takes a number which corresponds to the id assigned to the publisher. This is automatically assigned by the system to be the first available id in the system which is not already being used. This allows for id's which may have been freed up following the delete of a publisher.

Example value: 116

HTTP Methods

GET, PUT, PATCH, DELETE, HEAD, OPTIONS

Response Formats

Renders Formats XML, JSON, TEXT/CSV
Parses Formats XML, JSON, TEXT/CSV
Requires Authentication? Yes - for PUT, PATCH and DELETE requests

DELETE /publishers/144

Deletes the publisher located at /publishers/144 if the user meets the required permissions i.e. if they are the owner of the publisher/they created the publisher.

HTTP Status Code: 204 NO CONTENT or if the correct authorization token id not given then returns 403 FORBIDDEN

Example Input (for modifying a resource)

POST /fines

An example of the input needed for a POST request to /fines which sets the loan the fine is for and the amount is for.

HTTP Status Code: 201 CREATED or if no access token or incorrect access token 401 UNAUTHORIZED

Owner set to you automatically using the Token

Format: raw

Headers

Content-Type: application/xml;

<root>
    <loan>http://waisvm-rfp1.ecs.soton.ac.uk/users/3/loans/4</loan>
    <amount>0.10</amount>
</root>

Content-Type: application/json;

{
    "loan": "http://waisvm-rfp1.ecs.soton.ac.uk/users/3/loans/4",
    "amount": "0.10"
}

Content-Type: text/csv;

loan,amount
0/10,http://waisvm-rfp1.ecs.soton.ac.uk/users/3/loans/4

When an empty POST request is sent returns information about the fields that must be included in a POST HTTP request.

HTTP Status Code: 400 BAD REQUEST if required content is not sent or if it is not sent in the required format(s) or if the /users/3/loans/4 doesn't exist.

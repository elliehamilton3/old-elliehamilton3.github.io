Example Input (for modifying a resource)

POST /users

An example of the input needed for a POST request including some fields which can be used but are not required.

HTTP Status Code: 201 CREATED or if no access token or incorrect access token 401 UNAUTHORIZED

When a POST request is completed successfully and returns 201 CREATED response it returns a GET request on the newly created user which includes a link to the new user's loans page. No check exists in the system to check if the name already exists because its possible to have users in the system which have the same name, it is their user-id which makes them unique.

Owner set to you automatically using the Token

Format: raw

Headers

Content-Type: application/xml;

<root>
    <name>Bobby B</name>
</root>

Content-Type: application/json;

{
    "name": "Bobby B",
}

Content-Type: text/csv;

name
Bobby B
